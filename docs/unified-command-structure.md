# Unified command structure

## Overview

Rationalize operator-facing slash commands into a more discoverable and internally consistent structure. Current command affordances have drifted: some features are alias-heavy, some subcommands are hidden behind error messages, and related capabilities are split between slash commands, tools, and CLI subcommands. This node explores a unified command model with better discoverability, consistent subcommand grammar, and clear mapping to harness tools.
