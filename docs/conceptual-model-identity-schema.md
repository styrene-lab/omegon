# Conceptual Model Identity Schema — stable semantic model IDs independent of provider route

## Overview

Define canonical conceptual model IDs and metadata independent of serving provider route. These IDs support equivalence, capability grading, degradation, benchmarking, and grouped UX across direct providers, brokers, and subscription meta-providers.
