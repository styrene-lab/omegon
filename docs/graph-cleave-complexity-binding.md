# Bind web UI graph view to cleave complexity seed points — shared node values

## Overview

The design tree graph visualization in the web dashboard should use the same node values as the cleave algorithm's complexity seed points. When cleave_assess scores a task, those scoring nodes (systems touched, modifiers) should be visible in the graph view — making the complexity assessment legible to the operator.
