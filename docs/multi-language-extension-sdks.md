# Multi-Language Extension SDKs

## Overview

SDKs in other languages (Go, Python, TypeScript) that provide the same Extension trait interface. Developers can build extensions in their preferred language. Each SDK maintains version parity with omegon-extension crate and validates SDK version at install time.
