# Git URL-based Extension Installation

## Overview

Install extensions directly from git URLs (GitHub, GitLab, etc.). Supports both registry-published extensions and custom user forks. Format: `omegon install git:https://github.com/user/repo` or `omegon install git:user/repo` (shorthand).
