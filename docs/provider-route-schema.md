# Provider Route Schema — concrete auth/transport/model route serving a conceptual model

## Overview

Define concrete provider route records: provider, provider model ID, route model spec, conceptual model ID, auth surface, transport, route envelope, dialect, deprecation, and availability. Provider routes are execution identities that may serve shared conceptual models.
