# Extension Template Generator

## Overview

Scaffold new extensions with `cargo generate omegon-extension-template`. Generates boilerplate: Cargo.toml (with omegon-extension dep pinned to stable version), main.rs with Extension trait stub, manifest.toml template, README.md with publishing instructions.
